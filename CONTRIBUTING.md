# Contributing to OptiMin

Thank you for your interest in contributing.

OptiMin is a performance-focused modpack.  
The primary goal is **stability, minimalism, and measurable optimization**.

## Philosophy

Before suggesting anything, keep in mind:

- No gameplay content mods
- No feature bloat
- No unnecessary visual gimmicks
- Performance first, always
- Stability over novelty

Every addition must justify its existence technically.

## What You Can Contribute

You may contribute:

- Performance improvements
- Optimization mod suggestions (Fabric 1.20.1 only)
- Stability fixes
- Configuration tuning improvements
- Documentation improvements
- Benchmark comparisons

You should NOT propose:

- Content mods (biomes, mobs, items, etc.)
- Heavy visual overhauls
- Experimental or unstable mods
- Mods that duplicate existing functionality

## Mod Suggestion Guidelines

If suggesting a new mod, include:

- Clear explanation of what it optimizes
- Benchmark data (FPS, RAM usage, CPU usage)
- Compatibility confirmation with Sodium/Lithium stack
- Proof of active maintenance
- Fabric 1.20.1 support

Suggestions without measurable benefit may be declined.

## Pull Request Rules

Before opening a PR:

- Test on a clean installation
- Verify no crashes occur
- Verify no conflicts with existing mods
- Keep changes minimal and focused
- Do not modify unrelated files

Small, atomic pull requests are preferred.

---

## Reporting Issues

When reporting a bug:

- Include Minecraft version
- Include Fabric Loader version
- Include full mod list
- Provide crash report (with sensitive data removed)
- Describe exact reproduction steps

Incomplete reports may be closed.

OptiMin aims to stay clean, fast, and reliable.
Contribute with intention.
