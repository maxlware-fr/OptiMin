## 🚀 What is OptiMin?

OptiMin is a carefully curated **Fabric 1.20.1 optimization modpack** designed to:

- Maximize FPS
- Reduce memory usage
- Improve chunk & entity rendering
- Lower CPU load
- Maintain vanilla gameplay feel
- Stay minimal and stable

This modpack does **not** add gameplay content.  
It only improves performance, visuals, and user experience.

## 🎯 Target

- Low-end PCs
- Heavy multiplayer usage
- Large modded servers
- Players wanting smooth gameplay without gameplay changes

## ⚙️ Performance Stack

### Rendering Optimizations
- **Sodium** – Complete rendering engine rewrite for massive FPS gains
- **ImmediatelyFast** – UI & HUD rendering optimizations
- **More Culling** – Advanced block face culling
- **EntityCulling** – Prevents rendering unseen entities
- **FerriteCore** – Memory usage reduction
- **Lithium** – Game logic optimizations (AI, physics, ticking)
- **BadOptimizations** – Micro-optimizations across systems

### Background Resource Management
- **Dynamic FPS** – Reduces usage when game is unfocused
- **FPS Reducer** – Smart frame limiting when idle
- **ServerCore** – Server-side performance improvements

### Visual & UX Enhancements (Lightweight)
- 3D Skin Layers
- Chat Heads
- Chat Animation
- Jade (block/entity tooltips)
- JourneyMap
- JEI
- OptiGUI
- Simple Voice Chat
- Essential

### Core Dependencies
- Fabric API
- Fabric Language Kotlin
- Cloth Config
- Mod Menu
- Crash Assistant

## 🧠 Why Fabric?

Fabric was chosen because it:

- Is lightweight
- Loads faster than Forge/NeoForge
- Has lower overhead
- Provides the best optimization ecosystem (Sodium + Lithium + FerriteCore stack)

This ensures maximum compatibility and minimum performance impact.

## 💻 Recommended Settings

For best performance:

- Use 6–8GB RAM maximum (more is not always better)
- Enable VSync OFF
- Use fullscreen mode
- Reduce simulation distance if CPU bound
- Let Sodium handle advanced rendering settings

## 🔌 Server Compatibility

OptiMin is fully compatible with:

- Vanilla servers
- Fabric servers
- Most NeoForge/Forge servers (client-side mods only)

ServerCore improves performance when installed server-side.

## 🛠 Installation

1. Install .mrpack or .zip
2. Launch the game

## 🐞 Issues & Support

If you encounter crashes:

- Check the Crash Assistant report
- Verify mod version compatibility
- Avoid adding unknown optimization mods that conflict with Sodium
